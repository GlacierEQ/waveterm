## MCP (Model Context Protocol) Servers

### Core MCP Servers
- **Memory MCP Server** (Port 13000) - Persistent memory and context management
- **Filesystem MCP Server** (Port 13001) - Secure file system access and management
- **Terminal MCP Server** (Port 13002) - Terminal command execution and shell integration
- **Kubectl MCP Server** (Port 13003) - Kubernetes cluster management
- **Helm MCP Server** (Port 13004) - Helm chart management
- **Minikube MCP Server** (Port 13005) - Local Kubernetes development
- **Docker MCP Server** (Port 13006) - Docker container management

### MCP Server Capabilities

#### Memory MCP Server
- Persistent memory storage
- Context management
- Memory search and retrieval
- Session persistence

#### Filesystem MCP Server
- Secure file access (restricted to allowed paths)
- Directory browsing
- File content reading
- File search functionality
- Directory tree generation
- Security restrictions on dangerous operations

#### Terminal MCP Server
- Command execution with security filtering
- Command history tracking
- Shell integration
- Working directory management
- Dangerous command blocking
- Cross-platform shell support

#### Kubernetes Tools (kubectl, helm, minikube)
- Cluster management
- Resource deployment
- Chart management
- Local development environment
