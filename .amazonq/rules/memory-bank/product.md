# Wave Terminal - Product Overview

## Project Purpose
Wave Terminal is an open-source, AI-enhanced terminal that combines traditional command-line functionality with modern graphical capabilities. It bridges the gap between terminal-based workflows and visual interfaces, allowing developers to stay in their terminal environment while accessing web browsing, file previews, and AI assistance.

## Core Value Proposition
- **Unified Workflow**: Eliminates context switching between terminals, browsers, and AI tools
- **AI-First Design**: Integrated multi-agent system with 8 specialized agents for command analysis, security monitoring, and optimization
- **Enterprise-Ready**: Built-in security features, compliance monitoring, and Kubernetes integration
- **Cross-Platform**: Runs on macOS, Linux, and Windows with consistent experience

## Key Features & Capabilities

### AI & Intelligence
- **Multi-Agent System**: 8 specialized agents including command analysis, context management, security monitoring, and optimization
- **AI Chat Integration**: Support for OpenAI, Claude, Azure, Perplexity, and Ollama models
- **MCP Server Ecosystem**: 27+ specialized repositories with 47+ MCP servers for legal AI, forensics, and memory orchestration
- **Hyper-Intelligent Mode**: Auto-optimization, adaptive learning, and performance monitoring

### Terminal & Interface
- **Flexible Layout**: Drag & drop interface for organizing terminal blocks, editors, and browsers
- **Command Blocks**: Isolated command execution with monitoring and auto-close options
- **Rich File Preview**: Support for markdown, images, video, PDFs, CSVs, and directories
- **Built-in Editor**: Seamless remote file editing with syntax highlighting

### Connectivity & DevOps
- **One-Click Remote Connections**: Full terminal and file system access to remote servers
- **Kubernetes Integration**: Native kubectl, Helm, and Minikube support
- **WSH Command System**: Powerful workspace management and data sharing between sessions
- **Container Support**: Docker integration and container management

### Customization & Extensibility
- **Rich Theming**: Tab themes, terminal styles, and background images
- **MCP Server Framework**: Extensible server architecture for custom integrations
- **Configuration System**: Comprehensive settings and preset management

## Target Users

### Primary Users
- **DevOps Engineers**: Kubernetes management, container orchestration, infrastructure monitoring
- **Software Developers**: Code editing, debugging, AI-assisted development, remote development
- **System Administrators**: Server management, security monitoring, compliance tracking
- **Security Professionals**: Forensics tools, threat detection, audit capabilities

### Use Cases
- **Cloud-Native Development**: Kubernetes cluster management, microservices debugging
- **Remote Development**: Seamless remote server access with local-like experience
- **AI-Assisted Workflows**: Code generation, documentation, troubleshooting with integrated AI
- **Security Operations**: Real-time monitoring, forensics analysis, compliance reporting
- **Multi-Environment Management**: Managing multiple servers, containers, and cloud resources

## Technology Foundation
- **Frontend**: React 19, TypeScript, Electron for cross-platform desktop app
- **Backend**: Go 1.24.6 with comprehensive service architecture
- **AI Integration**: Multiple AI provider support with streaming capabilities
- **Database**: SQLite with migration system for local data storage
- **Build System**: Vite for fast development, Electron Builder for distribution
